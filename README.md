## Overview
This project is a responsive travel landing page designed for desktop view. It highlights visually appealing destinations and showcases the travel agency's offerings in an elegant and modern layout. The site is divided into three main sections and includes a footer.
## Live Demo
- [[Live Demo](https://travel-website-desktop-view-c4zq.onrender.com)]
## Project Features

### Section 1: Header
- Contains the navigation bar with a brand logo and a menu icon.
- Features a visually appealing header image composed of:
  - A blurry background.
  - An ellipse shape overlay.
  - A prominent traveler image.
- Includes a tagline: "Travel around the world."

### Section 2: Destination Highlights
- Highlights six travel destinations, each represented as a card.
  - Each card includes:
    - A destination image.
    - The destination name.
    - Location information (city and country).
- Encourages users to explore a wide variety of travel options.

### Section 3: Why Choose Us
- Contains three cards with key selling points:
  1. **Best Guide, Best Experience:** Focused on memorable trips with expert guides.
  2. **Special Tickets:** Highlights exclusive ticket offers for customers.
  3. **24/7 Support:** Promises round-the-clock support for travelers.
- Includes a call-to-action encouraging users to learn more about the services.

### Section 4: Footer
- Contains links to key pages:
  - Home
  - Destinations
  - Guide
  - Contact Us
- Social media icons (Instagram, Twitter, YouTube) for connecting with the brand.
- A brief description emphasizing the agency as "The best travel agent in the world" with a prompt to get in touch.

## Design and Styling
The page is styled using CSS and follows modern design principles:
- Font families:
  - "Gilda Display" for headings and key text.
  - "Urbanist" for body text.
- Colors:
  - Primary accent color: #13B0A8 (teal).
  - Background: White and soft gray shades.
- Responsive Design:
  - Ensures an optimized experience for desktop users.
  - Layout adapts seamlessly for varying screen resolutions.

## How to Run
1. Clone the repository to your local machine.
2. Open the `index.html` file in any modern web browser.
3. Ensure the `styles.css` file and the `images` folder are in the same directory as `index.html` to render the styles and images correctly.

## Folder Structure
- **index.html:** Contains the structure of the webpage.
- **styles.css:** Includes the styling rules for the webpage.
- **images/**: Contains all images used, including:
  - Destination images.
  - Icons (social media, location, arrows).
